Android Studio Hot Key

Windows Mac Function
⌘D 复制整行
⌘delete 删除整行
⌘X 剪切，若无选中文本剪切整行
⌥⇧↑↓ 上下移动整行
⌘⇧⏎ 光标换行
⌃P 光标移动到下一行
⌃N 光标移动到上一行
⌘[ 返回到上一次编辑位置
⌘] 前进到上一次编辑位置
⌘⌥⏎ 在当前行上添加一行，光标定位到行首
⌘⌥M 抽取为方法
⌘⌥F 抽取为成员属性
⌘⌥V 自动生成变量
⌘⌥P 将内部变量抽取成方法的参数
⌘／ 单行注释
⌘⌥／ 多行注释
⌥↑ 选择当前光标处所在单词或行
⌃O 快捷覆写方法
⌘Z 撤销上次操作
⇧F6 重命名
⌘⌥L 整理代码去除无效引用
⌘⇧U 大小写转换
⌘J 生成模板代码
⌘⌥T Surround With: if、while、try-catch、synchronized 等等
⌘PlusMinus 展开折叠代码块
⌘⌃O 查找工程内的文件或方法或字段
F4 定位到属性、方法、类等它们的声明
⌘F 查找 ⌘G 定位到下一个
⌘E 最近打开
⌃⌥H 方法调用
⌃H 类层级
⌘F12 方法结构
⌘U 父类同名方法
⌘⌃O 全屏切换
⌃R Build and Run
⌘, Android Studio Preferences
⌘; Project Structure
